# Инструмент анализа статистики продаж
### *Позволяет автоматизировать часть задач из отдела продаж*
## Принцип работы
### Выдает максимальную сумму продаж из массива данных
```java
  public int max() {
        int max = -1;
        for (int sale : sales) {
            if (sale > max) {
                max = sale;
            }
        }
        return max;
    }
```
## Использование
### Необходимо задать массив с данными по продажам
`int[] sales = {1, 2, 3, 4, 0};`
## Результат
`Максимальная сумма: 4`
